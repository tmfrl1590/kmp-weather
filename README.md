# KMP Weather App
### 소개
- Weather 앱을 KMP로 개발한 프로젝트

### 기능
- 메인페이지 : 검색된 도시의 현재 날씨 정보, 요일별 날씨 정보, 3시간 단위 날씨 정보
- 검색페이지 : 원하는 도시를 검색

### 기술스택
- Kotlin Multiplatform
- Compose Multiplatform 1.7.0
- Kotlin 2.0.21
- Ktor 3.0.0
- Koin 4.0.0
